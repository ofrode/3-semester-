# Лабораторная работа 7: Потоки ввода/вывода. Работа с файлами

## Описание

Реализован класс `FileReader` с перегруженным оператором `[]` для чтения символов из файла по указанной позиции.

## Структура проекта

- `hdr/FileReader.h` - заголовочный файл класса
- `src/FileReader.cpp` - реализация класса
- `src/main.cpp` - главный файл с меню
- `docs/` - папка для рабочих файлов

## Компиляция и запуск

### Использование Makefile:

```bash
make
./file_reader
```

### Или вручную:

```bash
g++ -std=c++17 -Wall -Wextra -c src/main.cpp -o obj/main.o
g++ -std=c++17 -Wall -Wextra -c src/FileReader.cpp -o obj/FileReader.o
g++ obj/main.o obj/FileReader.o -o file_reader
./file_reader
```

## Функциональность

Программа предоставляет меню с следующими возможностями:

1. **Создать/открыть файл** - создает новый файл или открывает существующий
2. **Записать строку в файл** - добавляет строку в конец файла
3. **Прочитать символ по позиции** - использует перегруженный оператор `[]` для чтения символа
4. **Показать размер файла** - отображает размер файла в байтах
5. **Выход** - завершение программы

## Пример использования

1. Запустите программу
2. Выберите опцию 1 для создания/открытия файла
3. Выберите опцию 2 для записи строки
4. Выберите опцию 3 для чтения символа по позиции
